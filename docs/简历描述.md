# 简历描述

项目名称：基于 RAG 与 MCP 的垂直领域智能知识库问答平台

项目描述：

基于 Python、FastAPI、LangChain、Hugging Face Embedding、Chroma、OpenWebUI 和 MCP 设计并实现垂直领域智能知识库问答平台，支持 PDF、Word、Markdown、TXT 等多类型文档上传、解析、分块、向量化存储和 Top-K 相似度检索。系统结合 RAG 技术将检索到的私有知识片段作为上下文输入大模型生成回答，并返回引用来源以提升回答可追溯性。同时封装 OpenAI-compatible API 接入 OpenWebUI，支持 mock、API 和 Ollama 三种模型调用模式，并基于 MCP Server 扩展知识库统计、本地文件读取、SQLite 笔记查询和知识库检索等工具调用能力。

技术栈：

Python、FastAPI、LangChain Text Splitters、Hugging Face sentence-transformers、Chroma、SQLite、RAG、OpenAI-compatible API、OpenWebUI、Ollama、Docker Compose、MCP。

项目亮点：

- 基于 Hugging Face sentence-transformers 构建文本向量表示，并使用 Chroma 实现本地持久化向量检索；
- 设计完整 RAG 流程，覆盖文档解析、文本切分、向量化、Top-K 检索、Prompt 组装和大模型生成；
- 封装 OpenAI-compatible API，使 OpenWebUI 能够作为统一前端入口调用自研 RAG 服务；
- 支持 mock、OpenAI-compatible API 和 Ollama 本地模型三种推理方式，提高部署灵活性；
- 基于 MCP Server 实现知识库统计、本地文件读取、SQLite 查询和知识库检索等工具调用能力；
- 实现知识库清空、文档删除、指定文档重建等管理能力，避免重复构建导致的向量冗余；
- 使用 Docker Compose 编排后端、OpenWebUI、Ollama 和 MCP Server 服务，提升项目部署和展示便利性。

个人工作：

- 接入 sentence-transformers，完成本地持久化向量检索和 sources 可追溯返回；
- 设计 mock-first 开发路径，保证无 API Key、无 Ollama、无 GPU 环境下仍可演示；
- 实现 OpenWebUI 兼容接口 `/v1/models` 和 `/v1/chat/completions`；
- 实现 MCP 工具服务，支持知识库统计、文件读取、notes 查询和知识库检索；
