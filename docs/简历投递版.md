# 简历投递版

## 简历项目写法

**基于 RAG 与 MCP 的垂直领域智能知识库问答平台**

基于 Python、FastAPI、Hugging Face sentence-transformers、Chroma、OpenWebUI 和 MCP 设计并实现垂直领域知识库问答平台，支持 PDF、Word、Markdown、TXT 等多类型文档上传、解析、分块、向量化存储和 Top-K 相似度检索。系统通过 RAG 技术将检索到的私有知识片段作为上下文输入大模型生成回答，并返回引用来源提升回答可追溯性；同时封装 OpenAI-compatible API 接入 OpenWebUI，支持 mock、API 和 Ollama 三种模型调用模式，并基于 MCP Server 扩展知识库统计、本地文件读取、SQLite 笔记查询和知识库检索等工具调用能力。

## 技术栈

Python、FastAPI、SQLite、Hugging Face sentence-transformers、Chroma、LangChain Text Splitters、RAG、OpenAI-compatible API、OpenWebUI、Ollama、Docker Compose、MCP。

## 项目亮点

- 基于 sentence-transformers 构建文本向量，并使用 Chroma 实现本地持久化向量检索；
- 设计完整 RAG 流程，覆盖文档解析、文本切分、向量化、Top-K 检索、Prompt 组装和大模型生成；
- 封装 `/v1/models` 和 `/v1/chat/completions`，使 OpenWebUI 能够作为统一聊天前端接入；
- 支持 mock、OpenAI-compatible API、Ollama 三种推理模式，保证无 API Key/无 GPU 环境下仍可演示；
- 实现知识库清空、文档删除、指定文档重建，避免重复构建造成向量冗余；
- 基于 MCP HTTP fallback Server 实现知识库统计、项目文件读取、SQLite notes 查询和知识库检索工具；
- 使用 Docker Compose 编排 backend、OpenWebUI、Ollama 和 MCP Server，并编写自动化测试脚本。

## 30 秒口述版

这个项目是一个企业知识库问答平台。用户上传文档后，系统会解析、切分、向量化并写入 Chroma，提问时先做 Top-K 检索，再把相关片段拼进 Prompt 调用模型回答。项目支持 mock、API 和 Ollama 三种模型模式，还提供 OpenAI-compatible API，所以可以直接接入 OpenWebUI。后面我又扩展了 MCP 工具服务，把知识库统计、文件读取、SQLite notes 查询和知识库检索封装成工具，方便智能体调用。整个项目用 FastAPI 和 Docker Compose 做工程化组织，并有测试脚本验证核心接口。

## 10 秒口述版
