MCP 是 Model Context Protocol，用于把外部系统和工具能力暴露给大模型应用或智能体客户端。

MCP Server 可以提供多种工具，例如本地文件读取、SQLite 查询、知识库统计、搜索服务调用、业务系统接口调用等。模型或智能体可以通过工具调用获得外部信息，而不只是依赖模型参数中的知识。

在企业知识库场景中，MCP 可以连接文档系统、数据库、工单系统、代码仓库和内部检索服务。这样 RAG 应用不仅能回答文档问题，也能进一步调用外部系统完成查询和自动化操作。

知识库统计可以返回当前文档数量和 chunk 数量。SQLite 查询只允许 SELECT，用于降低误操作风险。
